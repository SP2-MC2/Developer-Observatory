The building blocks of Jupyter notebooks are Cells. Click on New Cell under Cells to create your first one.

We create a cell to introduce our study. We use placeholders enclosed in percentage signs. %website% and %URL% are going to be replaced depending on the condition. %task1.description% dynamically references the second task since the first task is going to be this introduction and the tasks after that are going to change order.

Here we can see all cells we created so far. Next we create cells for our first programming task.

We want to introduce our first task with some markdown.

This is our first code cell, the heart of our programming task. Participants will be able to edit this code and execute it.

Now that we created some cells, we have to incorporate them into tasks.

Our introduction task only contains the single cell with our introduction markdown.

The first programming task consists of the two cells we created last.

You might want to come up with some more cells and tasks.

Time to create some conditions for our study.

The official python documentation is our first condition. The website and URL values are going to replace the %website% and %url% placeholders we used earlier in our introduction cell. Placeholders can also be used in code cells to e.g. import different libraries depending on condition.

Our second condition is Stack Overflow. Keep in mind this is just an example and do not think to much about the merit of this study.

The actual notebook files are created from notebook templates.

We set rules for the task order. We want our introduction to always come first, but the order of the tasks after that should be varied according to the Latin Squares method.

Click on Generate to create notebook files from the template.

Here we can select the conditions we want to use. Clicking on refresh shows us which files will be generated from the options we selected before we actually generate the files. The task order in each notebook is shown in brackets in the filename.

After generating the notebook file, it can be used with Jupyter.
